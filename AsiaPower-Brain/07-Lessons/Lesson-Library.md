# Lesson Library

记录：踩坑 → 修复 → 原因 → 经验。  
用于避免重复踩坑；不是事故八卦墙。

## 格式

用模板 `_templates/Lesson`。每条包含：

| 字段 | 说明 |
|------|------|
| 日期 | YYYY-MM-DD |
| 踩坑 | 发生了什么 |
| 修复 | 怎么修好的 |
| 原因 | 根因 |
| 经验 | 以后怎么做 / 禁止怎么做 |
| 相关 Decision | 可选链接 |
