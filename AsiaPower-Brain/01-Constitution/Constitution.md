# Constitution

不可破的规则。违反前必须走 [[11-Knowledge/CEO-Review/CEO-Review|CEO Review]]。

## 核心条

1. **正式名称：AsiaPower Brain** — 唯一长期知识库；Vault 路径 `AsiaPower-Brain/`。禁止用 wiki / 第二 Memory / 别名系统替代本 Brain。
2. **Obsidian 是唯一 Brain 载体** — 禁止再建第二套知识系统。
3. **Git 是唯一版本管理** — Vault 修改必须可追溯。
4. **Markdown Only** — 禁止数据库、私有格式、重插件依赖。
5. **Brain 不存代码** — 存 Decision / Architecture / Business / Mission / Lessons / Roadmap / Evidence Summary。
6. **Decision 有生命周期** — Proposed → Accepted → Superseded / Rejected；聊天 ≠ Decision。详见 [[06-Decisions/Decision-Log|Decision Log]]。
7. **Long-term Memory 只保存共识** — 不存日报、聊天、临时想法、完整 Evidence。详见 [[11-Knowledge/Long-term-Memory/Long-term-Memory|Long-term Memory]]。
8. **North Star 校准方向** — 见 [[00-Vision/North-Star|North Star]]；North Star ≠ Roadmap ≠ Production。
9. **Roadmap ≠ Production** — 路线图不等于已上线。
10. **Evidence 只存摘要** — 完整 ndjson 留在服务器，不进 Brain。
11. **生产部署硬红线** — commit → push GitHub → Release Manager；禁止常态 `--allow-dirty`。
12. **删数据 / force push / 对外群发** — 必须先问 CEO。
