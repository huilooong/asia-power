# AsiaPower Brain

Obsidian Vault · AsiaPower 唯一长期知识库（APBRAIN-001）。

## 打开方式

1. 安装 Obsidian
2. Open folder as vault → 选择本目录 `AsiaPower-Brain/`
3. 从 [[Home]] 开始导航

## 原则（短）

| 原则 | 含义 |
|------|------|
| Obsidian = 唯一 Brain | 不要第二套知识系统 |
| Git = 唯一版本 | 本目录在 AsiaPower 仓库内 |
| Markdown Only | 无 DB / 无私有格式 / 无重插件依赖 |
| 不存代码 | 存决策、架构、生意、教训、路线图、证据摘要 |
| Evidence | 只存 Summary；ndjson 留服务器 |
