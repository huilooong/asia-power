How to use
  1. Open the template in your browser — use the online links above, or locally: pages/quote.html in this project folder.
  2. Click any highlighted field to edit client name, items, prices and terms.
  3. Use + Add line for extra products; totals update automatically on the quotation.
  4. Print → Save as PDF to send to the client, or print on A4 paper.
  5. Fill in bank details in the footer before sending payment instructions.
