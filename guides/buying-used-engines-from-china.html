Buying Used Engines from China

A practical checklist for importers, workshops and parts wholesalers comparing used engines, half-cuts and engine-and-gearbox packages.

Start engine quote

1. Confirm the engine code before price comparison

The same vehicle model can use different engine codes across years, trims and markets. Ask for the code on the engine, VIN or chassis evidence, donor model, year and gearbox type before comparing offers.

For high-risk matches, separate the request into bare engine, complete engine with accessories, engine plus gearbox and full half-cut. These are different products and should not be compared as one price.

2. Ask what is included

PackageWhat to confirm
Bare engineLong block condition, visible damage, sensors or accessories excluded.
Complete engineStarter, alternator, compressor, harness, ECU and intake/exhaust parts where available.
Engine plus gearboxTransmission code, AT/MT/CVT, 2WD/4WD and donor compatibility.
Half-cutFront body cut, wiring, dashboard or suspension parts only if explicitly included.

3. Require photo evidence before payment

  • Engine bay or donor half-cut overview.
  • Engine-code marking or supplier confirmation photo.
  • Gearbox and accessories when the buyer needs a complete assembly.
  • Stock ID or reference number so the sales quote stays tied to one item.

4. Match logistics to the quotation

A low product price is not the final import cost. Ask whether the quote is EXW, FOB or CIF, which port is used, whether packing is included, and what documents can be provided for your destination country.

Fast buyer checklist

  • Engine code
  • Gearbox type and drivetrain
  • Required package
  • Quantity
