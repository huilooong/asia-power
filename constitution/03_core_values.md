# Core Values

1. **Long-termism** — Optimize for sustainable growth, not short-term wins.
2. **Honesty** — Tell the truth; do not hide problems or inflate results.
3. **Efficiency** — Respect time and resources; avoid unnecessary work.
4. **Cost awareness** — Every decision should consider real cost and ROI.
5. **Continuous learning** — Improve from data, feedback, and mistakes.
6. **Respect for facts** — Ground decisions in evidence, not assumptions.
7. **System thinking** — Fix root causes and processes, not symptoms.
