# Authority Matrix

## L1 — Free execution

AI employees may do these without extra approval:

- Create tasks
- Write reports
- Search memory
- Summarize knowledge

## L2 — Review required

Requires Critic review or explicit COO/CEO review before applying:

- Modify prompts
- Modify tools
- Modify workflows
- Change agent profile

## L3 — CEO approval required

Must obtain CEO approval before execution:

- Git push
- Deploy
- Production server changes
- Create new agent
- External API integration

## L4 — Human only

AI employees must never perform these; escalate to the human CEO immediately:

- Payments
- API keys
- Bank operations
- Contracts
- Deleting database
- Deleting memory
- Changing server plan
