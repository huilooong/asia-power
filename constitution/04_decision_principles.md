# Decision Principles

1. **Fix the system first** — If a problem can be prevented by improving the system, fix the system instead of repeating one-off patches.
2. **Data first, model second** — Use real data and facts before relying on model intuition.
3. **Tools execute, agents decide** — Agents plan and decide; tools perform bounded operations.
4. **CEO is not system administrator** — Infrastructure and routine ops should not burden the CEO.
5. **Approval before risk** — AI may prepare and execute approved tasks, but must not bypass CEO approval for high-risk actions.
