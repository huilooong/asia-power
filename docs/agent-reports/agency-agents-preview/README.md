## 各人设在讲什么（结论）

### 1. TikTok Strategist — 对口
- 3 秒钩子、FYP 算法、内容支柱 40/30/20/10（教/娱/励/推）
- 产出：选题框架、爆款公式、话题标签、创作者合作、跨平台改编
- **注意：** 默认受众偏 Gen Z/西方消费品牌；AsiaPower 要自己改成「西非汽配买家」场景

### 2. Short-Video Editing Coach — 对口（剪辑瓶颈）
- CapCut / PR / DaVinci / FCP 选型决策树
- 镜头语言、调色、音频、字幕、多平台导出、AI 辅助剪辑
- **注意：** 文里偏爱 CapCut + 提抖音生态；方法论可用，平台默认要换成 TikTok 国际版

### 3. SEO Specialist — 对口（品牌词/外链）
- 技术 SEO、内容集群、外链、SERP 特性、白帽规则
- 有强制「关键词互抢检查」（cannibalization）——对 asia-power.com 多类目页有用
- **注意：** 偏英文 Google SEO；不要用百度那套

### 4. Social Media Strategist — 半对口
- **主战场写的是 LinkedIn + Twitter 专业/B2B**，不是 FB/IG 消费号运营
- 适合：以后做 B2B 专业形象、内容日历总策划
- **不太适合：** 直接当「恢复 FB/IG 自动化」的说明书

### 5. Growth Hacker — 对口但薄
- 实验节奏、漏斗、获客渠道、CAC/LTV 指标清单
- 最短；适合碰撞 `growth-master-plan`，不会直接产出可执行脚本

### 6. Email Marketing Strategist — 半对口
- 强项：分群、生命周期序列、送达率、CRM↔ESP
- **偏 SaaS/房产式 nurture 流水线**（welcome 14 天、nurture 60–90 天）
- AsiaPower Track C 更像「网站询价 → 单封/少量跟进」；可借分群与合规规则，不要照搬长 nurture
