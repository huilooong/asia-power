# Agent Reports

This folder stores feedback from Codex, Cursor, and other agents.

Kongming / OpenClaw uses these reports to coordinate work, avoid duplicated instructions, and decide the next task.

Required latest reports:

- `codex-latest.md`
- `cursor-latest.md`

Archive individual reports in:

- `inbox/YYYYMMDD-HHMM-agent-task.md`

Reports must be factual. Do not hide failed commands, skipped tests, or uncertain assumptions.
