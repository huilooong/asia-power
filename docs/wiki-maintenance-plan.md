## 三、平台规则速查

### 3.1 维基百科（Wikipedia）

| 规则 | 含义（CEO 版） |
|------|----------------|
| **中立（NPOV）** | 像百科全书，不能写成广告；不用「行业领先」「最专业」 |
| **可验证（Verifiability）** | 每句重要事实都要能对应到脚注来源 |
| **显著性（Notability / NCORP）** | 必须有多篇**独立的、可靠的、二次**媒体报道（非官网、非通稿、非自家公众号）专门写你们公司 |
| **利益冲突（COI）** | 员工/老板不宜直接创建或大幅编辑自家词条；应走「请求创建条目」或请无关联编辑者代劳，并在讨论页披露关系 |
| **不能做什么** | 不能虚构引用、不能买软文当来源、不能把 Crunchbase/LinkedIn 当显著性依据 |

**当前评估：AsiaPower / BSB Motors / asia-power.com 均不满足维基显著性门槛。**  
检索未发现主流媒体、行业期刊对 asia-power.com 或 AsiaPower 汽配出口业务的独立深度报道。  
→ **建议：暂不创建英文维基词条**；先积累 2–3 篇独立报道后再评估。

参考政策页：
- https://en.wikipedia.org/wiki/Wikipedia:Notability_(organizations_and_companies)
- https://en.wikipedia.org/wiki/Wikipedia:Conflict_of_interest

### 3.2 百度百科

| 规则 | 含义（CEO 版） |
|------|----------------|
| **企业通道** | 大陆企业走 https://baike.baidu.com/enterprise/create ，用企业全称 + 统一社会信用代码验证 |
| **极速创建** | 从国家企业信用信息公示系统拉取基础工商信息，**24h 内可上线骨架词条** |
| **常规创建/扩充** | 需人工撰写 + 提交审核；账号需一定等级（常见要求：实名、编辑通过率） |
| **内容要求** | 客观中立；词条名 = 营业执照全称；禁止电话/微信/「第一/最好」等广告词 |
| **参考资料** | 权威第三方：央媒、省媒、政府公示、学术期刊。**官网、公众号、软文通常不能作核心参考** |
| **企业认证** | 认证后可认领词条、申请保护，减少恶意篡改 |

**当前评估：南阳芃芃信息科技有限公司 可走企业极速通道创建骨架词条；AsiaPower 品牌深度内容需补充第三方来源，否则扩充易被拒。**
