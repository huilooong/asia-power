请放到约定目录（建好后路径会写在这里），或发网盘/WhatsApp 链接均可：

## 设计原则（已定，不再争论）

| 环节 | 做法 |
|---|---|
| 剪辑 | 本地 ffmpeg 流水线 → 成片草稿 → 你人工确认 |
| 发布 | 优先官方 Content Posting API；**不做**浏览器自动发帖 |
| 评论 | 草稿 + 人工审核发送（子敬语气），不全自动 |
| 私信 | **不建新系统** → 视频导流到已训练的 WhatsApp 子敬 |
