# APSEO-011 Roadmap

## Mission

APSEO-011 is the 90-day Google Traffic Operating System for AsiaPower.

The goal is not more SEO documents.

The goal is measurable growth in:

- indexed pages
- keyword rankings
- organic clicks
- inquiries
- conversion

## Baseline From Existing Work

### Current Assets

| Asset | Current State | Use In APSEO-011 |
| --- | --- | --- |
| 50 generated engine pages | Complete basic SEO, sitemap, JSON-LD, FAQ, CTAs | Expand and optimize |

| Engine Intelligence V2 | Approved template architecture | Upgrade top pages |

### Core Constraint

AsiaPower must increase organic traffic without inventing official specs, claiming unverified stock, or creating duplicate canonical pages.

## 90-Day Roadmap

### Phase 1: Foundation And Indexing Control

Timeline:

```text
Days 1-15
```

Objective:

Increase crawl clarity and make the first 50 pages measurable.

Workstreams:

1. Canonical cleanup plan for duplicate legacy/code-first engine URLs.
2. Search Console property review and sitemap validation.
3. Index coverage baseline for all engine pages.
4. Top 16 engine pages selected for V2 optimization.
5. First Ghana/Nigeria country landing page briefs.
6. Internal link map from homepage, engine catalog, brand pages, and half-cut pages.

### Phase 4: Search Console Iteration And Conversion

Timeline:

```text
Days 76-90
```

Objective:

Use real Google data to improve pages that have impressions but weak clicks or weak conversion.

Workstreams:

1. Pull Search Console queries for engine pages.
2. Identify pages with impressions but low CTR.
3. Rewrite titles/meta for top 20 low-CTR pages.
4. Identify indexed pages with no clicks and add internal links or content depth.
5. Identify inquiry pages with organic visits but no inquiry and improve CTA placement.
6. Feed winning keywords into APCGO daily opportunity database.

Measurable outcomes:

| KPI | Target |

| Search Console query review cycles | 3 |
| Low-CTR pages optimized | 20 |
| No-click indexed pages improved | 20 |

## Schema Expansion Roadmap

### Phase 1 Schema

Required on all improved engine pages:

- `WebPage`
- `BreadcrumbList`
- `Product`
- `FAQPage`

### Phase 2 Schema

Add where visible on page:

- `ItemList` for related engines
- `ItemList` for related half-cuts
- `ItemList` for related vehicles
- `HowTo` for inspection or export process

### Safety Rules

Do not add:

- fake price
- fake review
- fake rating
- guaranteed availability
- unverified official specifications

## Search Console Roadmap

### Weekly Search Console Workflow

1. Export page-level data for engine, country, guide, and comparison URLs.
2. Export query-level data for top pages.
3. Mark queries by cluster:
   - engine code
   - vehicle
   - country import
   - half-cut
   - gearbox
   - comparison
   - buyer question
4. Identify:
   - impressions but no clicks
   - high CTR but low position
   - indexed pages with no impressions
   - pages receiving irrelevant queries
5. Update pages based on data.
