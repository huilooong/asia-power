# APSEO-011 Execution Plan

## Objective

This execution plan converts the existing AsiaPower SEO architecture into a 90-day operating plan.

Every task must increase at least one measurable output:

- indexed pages
- keyword rankings
- organic clicks
- inquiries

## 3. Country Landing Page Strategy

### Priority Markets

APSEO-011 focuses first on:

```text
Ghana + Nigeria
```

Reason:

- APCGO already validated company and keyword opportunities.
- Both markets have engine, gearbox, half-cut and fleet demand.
- Country pages can convert buyers who do not know exact engine code.

### Page Structure

Each country page must include:

- country-specific H1
- engine/gearbox/half-cut use case
- top engine families
- import and quote process
- inspection and confirmation policy
- related engine pages
- related vehicle pages
- WhatsApp CTA
- inquiry form
- FAQ
- JSON-LD

### Weekly Query Actions

| Finding | Action | KPI |
| --- | --- | --- |

| Ranking positions 8-20 | Add content depth and internal links | rankings |

- engine_code
- vehicle
- country
- half_cut
- gearbox

- export
- brand

## 7. Index Coverage Improvements

### Weekly Checks

Check:

- submitted vs indexed
- discovered but not indexed
- crawled but not indexed
- duplicate without user-selected canonical
- alternate page with proper canonical
- soft 404
- redirects
- blocked by robots

### Remediation

| Issue | Fix |
| --- | --- |
| Crawled but not indexed | Add unique content depth and internal links |
| Discovered not indexed | Add stronger hub links and sitemap lastmod |
| Duplicate canonical | consolidate to code-first engine URL |
| Soft 404 | add content depth or remove from sitemap |
| Low-value page | upgrade or noindex until useful |

### Cluster Rule

Do not create a new page if an existing page can satisfy the query with one high-quality section.

Create a new page only when:

- intent is different,
- query has commercial value,
- internal links can support it,
- inquiry path is clear.
