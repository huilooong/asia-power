# SEO Plan

## Intent split

| Intent | Title center | H1 center | Body / modules |
|--------|--------------|-----------|----------------|
| Half-Cut | Vehicle + Half Cut + engine confirm | Vehicle Half Cut — engine specs | Stock facts, year, status |
| Engine | Brand + Code + Disp + Fuel + Engine | Same as title entity | Compatible Vehicles long-tail |

## Examples (templates; fill only from real fields)

### Half-Cut

- Title: `Toyota Camry Half Cut 2AZ-FE 2.4L | AsiaPower`  
- H1: `Toyota Camry Half Cut — 2AZ-FE 2.4L Petrol`  
- Meta: vehicle + year + engine confirm + availability language（无货不写 Available）

### Engine

- Title: `Toyota 2AZ-FE 2.4L Petrol Engine Supplier | AsiaPower`  
- H1: `Toyota 2AZ-FE 2.4L Petrol Engine`  
- Body naturally: Camry 2AZ-FE engine / RAV4 2AZ-FE engine via Compatible Vehicles — **not** stuffed into H1

## Sync checklist (implementation later)

title · H1 · meta description · og:title · og:description · JSON-LD name/description（若已有）

## Hard rules

1. No canonical logic change  
2. No duplicate thin URLs  
3. No bulk vehicle page generation  
4. No “guaranteed compatibility” in SEO copy  
5. Missing field → degrade string, never invent  

## Future (document only — not build)

Engine Detail → Vehicle pages when they exist; Half-Cut ↔ Engine bidirectional links on existing URLs only.
