2

账号实名 = 企业

若阿里云账号还是个人 → 账号中心 → 实名认证 → 升级为企业，名称填 南阳芃芃信息科技有限公司。

3

付款 · 填域名持有者（关键！）

付款后进入 域名控制台 → 信息模板：

  1. 持有者类型：企业
  2. 持有者名称：南阳芃芃信息科技有限公司（与执照完全一致）
  3. 统一社会信用代码：从营业执照复制
  4. 上传营业执照 → 提交 域名实名认证

⚠️ 不要填个人姓名，否则企业备案会被拒。

4

等实名通过 · 下载域名证书

通常 1～3 天。状态变「正常」后 → 域名详情 → 下载域名证书 PDF（备案要用）。

5

开始企业备案

打开 beian.aliyun.com → 开始备案 / 首次备案。

  1. 主体：企业 · 南阳芃芃信息科技有限公司
  2. 上传：营业执照 + 域名证书 + 法人身份证
  3. 网站名可填 AsiaPower；服务器在境外按页面提示选
  4. 法人手机刷脸（若提示）

「备案」= 在工信部登记这个域名归哪家公司；企业微信靠这个校验。
