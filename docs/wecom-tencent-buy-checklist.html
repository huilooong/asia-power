Lighthouse 是什么？（1 分钟看懂）

Lighthouse = 轻量应用服务器，可以理解为「简化版云服务器」：CPU+内存+硬盘+带宽打包成固定套餐，控制台更简单，适合跑网站、小程序后台、企微回调这类轻量任务。

和「云服务器 CVM」比：CVM 像自己配电脑（选项多、贵、偏专业）；Lighthouse 像买套餐机（选项少、便宜、开箱即用）。我们备案用 Lighthouse 就够，不必买 CVM。

8 步购买流程（腾讯云）

1

注册腾讯云 + 企业实名

打开 cloud.tencent.com → 注册/登录 → 账号中心 → 实名认证 → 企业认证。

企业名称填 南阳芃芃信息科技有限公司（与营业执照完全一致）。

「实名认证」= 证明这个腾讯云账号属于哪家公司，备案时必须企业实名。

2

买域名 asia-power.cn（+ 可选 asia-power.com.cn）

打开 dnspod.cloud.tencent.com（腾讯云域名注册）→ 搜索 asia-power.cn → 加入购物车 → 结算。

建议同时加购 asia-power.com.cn 做品牌保护（可选，多 ¥8.8～33/年）。

购买时长建议 1～3 年；结算页以实际价格为准（新用户 .cn / .com.cn 可能 ¥8.8）。

企微回调只需 1 个域名备案；第二个域名只注册、不解析也行。

3

域名持有者 = 公司（关键！）

付款后 → 域名控制台 → 信息模板 / 实名认证：

  1. 持有者类型：企业
  2. 持有者名称：南阳芃芃信息科技有限公司
  3. 统一社会信用代码：从营业执照复制（18 位）
  4. 上传营业执照 → 提交 域名实名认证

⚠️ 不要填个人姓名，否则企业备案会被拒。

4

买轻量应用服务器 Lighthouse（同账号）

按上方 「购买点击路径 A」 操作：活动页 轻量云专场特惠（不是 OpenClaw 页、不是产品原价页）→ 企业专区/同价续费 → 2核2G ¥99/年 → 上海或广州 → 1年 → 付款。

也可选活动页「组合购」— 域名+服务器一起买更省事。

「备案接入」= 备案时要登记网站托管在哪台服务器；买国内 Lighthouse 审核最顺。企微回调极轻，2核2G 绰绰有余。

5

等域名实名通过 · 下载域名证书

通常 1～3 天。状态变「正常」后 → 域名详情 → 下载域名证书 PDF（备案要用）。

域名实名信息需入库工信部，建议实名通过后再等 3 天 再提交备案（腾讯云页面会提示）。

6

提交企业 ICP 备案

打开 icp.cloud.tencent.com → 开始备案 / 首次备案。

  1. 主体：企业 · 南阳芃芃信息科技有限公司
  2. 上传：营业执照 + 域名证书 + 法人身份证
  3. 网站名称可填 AsiaPower 或公司简称
  4. 接入信息：选刚买的 Lighthouse 轻量服务器（自动带出 IP）
  5. 法人手机刷脸（若提示）

「备案」= 在工信部登记这个域名归哪家公司；企业微信靠这个校验。

7

等管局审核（7～20 工作日）

腾讯云初审 → 短信核验 → 管局审核。通过后可在 beian.miit.gov.cn 查到备案号，主体应为 南阳芃芃信息科技有限公司。

8

发给技术 · 等配置回调

备案通过后，把以下信息发给技术（勿发密码 / Secret / Token）：

  1. 买到的域名（主域名 asia-power.cn；若买了保护域名也一并告知）
  2. 腾讯云域名控制台截图（持有者 = 公司）
  3. 备案号截图（beian.miit.gov.cn 可查）
  4. Lighthouse 公网 IP

技术会做：DNS → 服务器 → nginx → 企微后台改 URL。您最后只需在企微后台点保存。
