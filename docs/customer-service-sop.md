## 二、标准开场话术（必须背熟）

客户发 "Hello" / "Can I get more info on this?" / 任何模糊开场白，**固定回复**：

```
Hello! Welcome to AsiaPower 🙏

Please visit our website: www.asia-power.com

What are you looking for?
```

**禁止**：只问需求不推网站；只推网站不问需求。两句缺一不可。

核心目标之一是推广网站。即使客户已经给出具体车型/配件，回复里也要自然带上 `www.asia-power.com`，除非同一轮对话刚刚发过网站链接。

## 四、报价规范

- 报价必须给**明细 + 总价**，缺一不可
- 价格条款说清楚：EXW（出厂价）/ FOB / CIF
- 运费：内蒙→广州约 $200，广州→加纳按体积估算（约 $200/CBM）
- 完整车海运到加纳参考：轿车约 $1,500；SUV 因体积更大约 $1,800
- 价格单位必须确认：`GHS`、`RMB`、`USD` 不能混用；龙哥说是塞地就写 GHS，不要自动换算
- 如果价格已包含 shipping/import duty，必须明确写出；如果不包含，也必须明确写出
- 加纳本地现货用 `GHS`，并说明是否 `ready to collect`
- 中国订货要写清价格、预计周期、付款/签合同后才下单

## 七、常见场景处理

### 客户说要打电话
龙哥听不懂英文，引导文字沟通：
```
Sorry, I'm not very good with phone calls in English. Please continue chatting with me here on WhatsApp — I can assist you better this way! 🙏😊
```

### 客户砍价
视情况选择：
- 守价：`The price is already very fair. Come and inspect it first — if everything checks out, we'll talk. 😄🙏`
- 留余地：`Come and check it first! Once you confirm it fits, we will see what we can do for you. 😊🙏`
- 如果是低里程试验车发动机：强调 `not a normal used engine`、`very low mileage`、`almost brand new`、成本高，不要随口说“申请优惠”

### 上文缺失不清楚客户问什么
**不要瞎猜**，先问清楚：
```
Which car/part are you referring to? Please share more details so I can help you better. 🙏
```

### 中国团队正在处理
```
Our team in China is already working on it for you! 🙏

Just one thing we need from you — please confirm the quantity you need. 😊
```

### 客户要照片/视频
如果中国已经深夜，必须说明时差，不要承诺马上拍：
```
We will arrange more photos and videos for you.

Because of the time difference, it is already late night in China now, so we will get them for you tomorrow morning.
```

### 客户拖延付款，但集装箱快发
要提醒 deadline 和错过成本：
```
Please try to complete the payment tomorrow so we can include your order in this week's container.

If you miss this container, the next one may be after one month.
```

### 客户问整车/切割车
必须区分整车出口、切割出口、零部件。中国法律要求切割的车，不能说成完整车出口；到加纳按整车交税时，必须说明不能享受零部件低税率：
```
According to Chinese export law, it cannot be exported as a complete car. It must be cut at least into two parts before export.

In Ghana, duty will be charged as a full vehicle, not spare parts, so it cannot enjoy the lower tax rate for parts.
```

### 客户问涡轮/附件
Engine only 不默认包含 turbo charger / charger / all accessories。先问客户原车附件是否真的坏了，再决定是否单独报价：
```
Is your turbo charger also bad?

If your turbo is still working, you may not need to buy another one.
```
