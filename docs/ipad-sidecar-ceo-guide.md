# iPad 当 Mac 第二块屏幕（Sidecar）— CEO 指南

## 结论

**不用装任何 App。** 把 iPad 当成 Mac 的「扩展显示器」，Filco 键盘 + 鼠标继续在 Mac 上操作，窗口可以拖到 iPad 上显示。

| 方案 | 一句话 |
|------|--------|
| **Sidecar（本方案）** | iPad 显示 **Mac 的桌面**，当第二块屏幕用 |
| **Universal Control** | 一套键鼠同时控制 Mac **和** iPad，iPad 仍显示 **自己的 iPad 桌面** |
| **Tracepad 等第三方** | 需装 App，把 iPad 当触控板 — **你已放弃此方案** |

---

## 前提条件（先核对）

| 项目 | 要求 |
|------|------|
| Apple 账号 | Mac 和 iPad **登录同一个 Apple ID**，且已开 **双重认证** |
| 网络 | 两台设备 **同一 Wi‑Fi**；**蓝牙**、**接力（Handoff）** 均打开 |
| 距离 | 无线使用时相距 **10 米以内** |
| iPad 型号 | iPad Pro 全系；iPad 第 6 代及更新；iPad mini 第 5 代及更新；iPad Air 第 3 代及更新 |
| Mac 型号 | 2016 年及之后大多数 Mac（详见 [Apple 官方要求](https://support.apple.com/en-us/102597)） |
| 系统 | Mac **macOS 10.15 及以上**（你当前 26.5.1 ✅）；iPad **iPadOS 13 及以上** |
| 额外注意 | iPad **不要开个人热点**；Mac **不要开互联网共享**；VPN 若阻断本地网络可能影响连接 |

**无需安装任何第三方 App**，系统自带 Sidecar。

---

## 连接步骤（5–8 步）

| 步骤 | 在哪操作 | 做什么 |
|:----:|----------|--------|
| **1** | Mac | 确认 Mac、iPad 已登录 **同一 Apple ID** |
| **2** | Mac | **系统设置 → 通用 → 隔空播放与接力**：打开 **接力** |
| **3** | iPad | **设置 → 通用 → 隔空播放与接力**：打开 **接力**；**设置 → 蓝牙** 打开 |
| **4** | Mac | **系统设置 → 显示器** → 点 **+** 或顶部出现的 **iPad 名称** → 选 **「使用作副屏」** |
| **5** | Mac（备选） | 菜单栏点 **屏幕镜像**（控制中心里）→ 选你的 iPad → **用作副屏** |
| **6** | Mac | 连接成功后，iPad 会显示 Mac 桌面延伸出来的区域 |
| **7** | Mac | 用 Filco 键盘 + 鼠标：把窗口 **往 iPad 方向拖**，越过屏幕边缘即可放到 iPad 上 |
| **8** | Mac（可选） | **系统设置 → 显示器 → 点 iPad** → 可关侧边栏 / Touch Bar、调排列位置 |

**更快方式：** 把鼠标移到 Mac 窗口左上角 **绿色全屏按钮**，悬停后选 **「移动到 iPad」**。

**更稳方式：** 用 iPad 原装 USB 线连 Mac，可充电且延迟更低（iPad 需信任此 Mac）。

---

## 扩展 vs 镜像

| 模式 | 效果 | 怎么选 |
|------|------|--------|
| **扩展（推荐）** | Mac 和 iPad **各显示不同区域**，桌面变宽，可拖不同窗口到 iPad | 系统设置 → 显示器 → iPad → **「扩展显示器」** |
| **镜像** | iPad **复制** Mac 主屏，两边显示一样 | 菜单栏 **屏幕镜像** → 选 iPad → **「镜像内建 Retina 显示器」** |

**CEO 日常建议：用「扩展」** — Mac 主屏放主工作，iPad 放参考文档、微信、表格等。

## 窗口往哪拖？

```
┌─────────────┐     ┌─────────────┐
│   Mac 主屏   │ ──→ │  iPad 副屏   │
│  (Filco键鼠) │     │  (只显示)    │
└─────────────┘     └─────────────┘
        ↑ 鼠标往右拖过边缘，窗口就进 iPad
```

- 在 **系统设置 → 显示器** 里可拖动两块屏幕图标，调整 iPad 在左还是右（决定往哪边拖）。
- 拖回 Mac：同样往反方向拖，或绿色按钮选 **「移回 Mac」**。

## macOS 26 / iPadOS 26 须知

| 项目 | 说明 |
|------|------|
| Sidecar 本身 | **仍可用**，无需额外 App |
| 无线 vs 有线 | 无线方便；**USB 线更稳、更省电** |
| Sidecar 期间切 iPad App | 若你在 iPad 上打开 **iPad 自己的 App**，Sidecar 会 **暂停**；点 iPad 主屏上的 **Sidecar** 图标可恢复 |
| iPadOS 26.0 | 曾暂时去掉 Slide Over（与 Sidecar 无关）；**26.1+ 已恢复** |
| VPN | 部分 VPN 会挡本地网络，Sidecar 连不上时可 **暂时关 VPN** 试一次 |

## 常见问题

| 现象 | 处理 |
|------|------|
| 找不到 iPad | 确认同一 Apple ID、同一 Wi‑Fi、蓝牙 + 接力已开；两台设备都 **重启** 一次 |
| 列表里没有「用作副屏」 | 查 iPad/Mac 型号是否支持；把 **macOS、iPadOS 更到最新** |
| 连上但卡顿 | 改 **USB 线** 连接；路由器旁使用；关 iPad 个人热点 |
| 鼠标拖不过去 | **系统设置 → 显示器** 里确认 iPad 排列位置（左/右） |
| 和 Universal Control 搞混 | Sidecar = iPad 显示 **Mac 桌面**；Universal Control = iPad 仍显示 **iPad 桌面**，键鼠共用 |
| 想断开 | 菜单栏 **屏幕镜像** → 断开；或 iPad 侧边栏点 **断开** |
