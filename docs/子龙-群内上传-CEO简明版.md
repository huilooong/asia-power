# 子龙 · 群内上传 — CEO 必读（简明版）

> 完整技术文档：`data/knowledge-base/wecom-zilong-training-runbook.md`（给 Agent 看）

---

## 一句话结论

客户在**企业微信群**发图 + 说「子龙032 上传」→ 子龙检查 → **您必须在审核页点确认** → 才会上网站。

**不会自动发布。** 没点确认，不会上线。

---

## 客户在群里怎么用

1. 在供应商群里直接发车辆照片（可多张）
2. @AsiaPower 库存 Agent 说：**「子龙032 上传」**（032 换成 Excel 行号）
3. 子龙回复检查结果（VIN、发动机、有没有问题）
4. **您**打开审核页确认 → 才会提交到网站

---

## 您每天要做的事

### ① 打开审核页

双击：

`AsiaPower/docs/打开QXB上传进度.command`

或终端运行后浏览器打开：**http://127.0.0.1:8789/review**

### ② 看群里子龙回复

- 有阻塞项 → 在审核页处理或让客户补图
- 检查通过 → 点「确认上传」

### ③ 配置供应商群（新群才要做）

1. 从企微回调日志找到群 **ChatId**（形如 `wrXXXX`）
2. 复制 `config/wecom-group-suppliers.json.example` → `config/wecom-group-suppliers.json`
3. 填入 ChatId 和供应商名称
4. `.env` 里 `WECOM_ALLOWED_CHAT_IDS` 加上这个 ChatId

### ④ 企微管理后台

应用 **AsiaPower 库存 Agent** → 接收消息 → 必须勾选 **图片**（不只文本）

## 常见问题

| 现象 | 怎么办 |
|------|--------|
| 发图没反应 | 检查群是否在白名单；后台是否勾选图片消息 |
| 子龙说 Excel 无此行 | 核对汽修宝导出的行号 |
| 想立刻上线 | 必须先过审核页 8789，再 Admin 批准 |
