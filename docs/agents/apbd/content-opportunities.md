# APBD Content Opportunities

## Purpose

APBD must deliver 20 new content opportunities every day.

The goal is not content volume.
The goal is traffic, leads, and inquiries.

Every content opportunity must connect to a buyer search, a commercial need, and a possible AsiaPower product path.

## Required Fields

Each content opportunity must include:

```text
Target keyword
Search intent
Estimated business value
Expected inquiry value
Priority
Recommended page type
Reason
Next action
```

## Page Types

APBD may recommend:

- engine pages
- vehicle pages
- country pages
- buying guides
- FAQ
- comparison pages
- gearbox matching pages
- half-cut pages
- export process pages

## Content Priority

### S Priority

High traffic or high buyer intent, and AsiaPower can serve the demand.

Examples:

- `G4KD engine for sale Ghana`
- `2TR-FE engine half cut`
- `1KD-FTV engine import Africa`
- `engine and gearbox from China`

### A Priority

Strong commercial intent but may need more data or inventory support.

Examples:

- vehicle-specific engine pages
- country landing pages
- comparison pages

### B Priority

Useful but less urgent.

Examples:

- broad educational FAQ
- low-volume engine code
- early market exploration topic

## Opportunity Categories

### Engine Pages

Purpose:

- capture engine-code searches
- generate quote inquiries
- connect to half-cut and gearbox supply

Example:

```text
G4KE engine for sale
```

### Vehicle Pages

Purpose:

- capture buyers who search by vehicle model, not engine code

Example:

```text
Hyundai ix35 engine replacement
```

### Country Pages

Purpose:

- capture import/export buyers in specific markets

Example:

```text
used engines import Ghana
```

### Buying Guides

Purpose:

- build trust and educate buyers

Example:

```text
how to buy used engine from China
```

### FAQ

Purpose:

- capture long-tail buyer questions

Example:

```text
what is a half cut engine
```

### Comparison Pages

Purpose:

- capture buyers deciding between options

Example:

```text
1KD-FTV vs 2KD-FTV
```

## Daily Output Format

Daily content opportunity files should use:

```text
docs/agents/apbd/daily-content-opportunities-YYYY-MM-DD.md
```

Recommended table:

```text
| Target Keyword | Search Intent | Estimated Business Value | Expected Inquiry Value | Priority | Page Type | Reason | Next Action |
```

## Quality Standard

Every recommendation must be actionable.

Good:

```text
Target keyword: G4KD engine Ghana
Search intent: buyer wants engine supply or quote
Business value: high because Hyundai/Kia engines are common in West Africa
Inquiry value: high
Priority: S
Page type: Engine Intelligence page
Next action: improve G4KD canonical page with Ghana export CTA
```

Bad:

```text
Write more blogs about engines.
```

## Human Approval

APBD recommends content.

Humans approve:

- page creation
- page update
- publishing
- distribution
- deployment

APBD must not deploy pages automatically.
